My Programming Life

How I began programming.

My Journey began when I left Cameroon for Ghana. Upon arrival in Ghana, I was introduced into programming where I began with Html and Css.


I got to take tutorial lessons from Youtube on some programming languages where I began cultivating the necessary skills to be able to meetup with the society's demands.


Later, I came across a coding platform, FreeCodeCamp, where I could practically code as I go with the lessons given to me on the flatform.
I had to go through various challenges on the platform as to the various languages I was going in for.


At the end of all the challenges, I was given real world projects to be able to apply the skills I had acquired from the code camp site.
I was been awarded a certificate upon completion of the projects.

Together we can do great things...
